# Postup tvorby stránky na téma Air Jordan (vygenerováno pomocí AI)

Tato stránka byla vytvořena pomocí umělé inteligence a obsahuje základní informace o značce **Air Jordan**, její historii a obuvi.

## 1. Vytvoření HTML souboru

HTML soubor obsahuje následující základní sekce:
- **Header**: Titulní část s názvem stránky a úvodním textem o Air Jordan.
- **Sekce o obuvi**: Textový popis obuvi Air Jordan.
- **Postup**: Odkaz na tento soubor s popisem kroků tvorby stránky.
- **Footer**: Zápatí se základními informacemi.

HTML kód vytvořený AI zahrnoval základní strukturu a propojení s CSS souborem pro styling.

## 2. Vytvoření CSS souboru

Pro zajištění vzhledu stránky byl použit jednoduchý CSS soubor, který upravuje stylování textu, nadpisů a layoutu. Cílem bylo vytvořit čistý a přehledný design.

## 3. Odkaz na postup

Celý postup tvorby stránky, včetně podrobného popisu kroků, je obsažen v tomto souboru, aby uživatelé mohli lépe pochopit, jak stránka funguje a jak byla vytvořena.

## 4. Co bylo použito

- **HTML**: Základní struktura webové stránky.
- **CSS**: Styling pro vzhled stránky.
- **AI**: Vygenerovalo kód a postupy pro tvorbu stránky.

Tato stránka je jednoduchým příkladem použití HTML a CSS pro vytvoření statické webové stránky zaměřené na značku Air Jordan.
